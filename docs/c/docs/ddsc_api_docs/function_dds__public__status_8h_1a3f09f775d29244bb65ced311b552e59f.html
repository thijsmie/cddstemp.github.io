Function dds_get_inconsistent_topic_status¶

Function Documentation¶

dds_return_t dds_get_inconsistent_topic_status(dds_entity_t topic, dds_inconsistent_topic_status_t *status)¶

Get INCONSISTENT_TOPIC status.

This operation gets the status value corresponding to INCONSISTENT_TOPIC and reset the status. The value can be obtained, only if the status is enabled for an entity. NULL value for status is allowed and it will reset the trigger value when status is enabled.

Parameters

  • [in] topic – The entity to get the status

  • [out] status – The pointer to DCPS_Status_InconsistentTopic to get the status

Returns DDS_RETCODE_ERRORDDS_RETCODE_ERROR

An internal error has occurred.

Returns DDS_RETCODE_BAD_PARAMETERDDS_RETCODE_BAD_PARAMETER

One of the given arguments is not valid.

Returns DDS_RETCODE_ILLEGAL_OPERATIONDDS_RETCODE_ILLEGAL_OPERATION

The operation is invoked on an inappropriate object.

Returns DDS_RETCODE_ALREADY_DELETEDDDS_RETCODE_ALREADY_DELETED

The entity has already been deleted.

Returns

0 - Success

Returns

<0 - Failure
